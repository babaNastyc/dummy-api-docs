# API Endpoints

Complete reference for all available endpoints in the Dummy API.

## Status

### Get API Status

Check the current status and version of the API.

<ParamField body="none" path="none" query="none">

</ParamField>

<ResponseField name="status" type="string">
  Current status of the API
</ResponseField>

<ResponseField name="message" type="string">
  Status message
</ResponseField>

<ResponseField name="version" type="string">
  API version number
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET https://api.example.com/v1/status \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/status', {
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});

const data = await response.json();
```

</CodeGroup>

**Response:**

```json
{
  "status": "ok",
  "message": "API is running",
  "version": "1.0.0"
}
```

## Resources

### List Resources

Retrieve a list of all resources.

<ParamField body="none" path="none" query="page,limit">

<ParamFieldObject name="page" type="integer" required>
  Page number for pagination (default: 1)
</ParamFieldObject>

<ParamFieldObject name="limit" type="integer" required>
  Number of items per page (default: 10, max: 100)
</ParamFieldObject>

</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/v1/resources?page=1&limit=10" \
  -H "Authorization: Bearer your_api_key"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/resources?page=1&limit=10', {
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});

const data = await response.json();
```

</CodeGroup>

**Response:**

```json
{
  "data": [
    {
      "id": "res_123",
      "name": "Resource 1",
      "type": "example",
      "created_at": "2025-01-01T00:00:00Z"
    }
  ],
  "meta": {
    "page": 1,
    "limit": 10,
    "total": 1
  }
}
```

### Create Resource

Create a new resource.

<ParamField body="name,type" path="none" query="none">

<ParamFieldObject name="name" type="string" required>
  Name of the resource
</ParamFieldObject>

<ParamFieldObject name="type" type="string" required>
  Type of the resource
</ParamFieldObject>

</ParamField>

<CodeGroup>
```bash cURL
curl -X POST https://api.example.com/v1/resources \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Resource",
    "type": "example"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/resources', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Resource',
    type: 'example'
  })
});

const data = await response.json();
```

</CodeGroup>

**Response:**

```json
{
  "data": {
    "id": "res_123",
    "name": "My Resource",
    "type": "example",
    "created_at": "2025-01-01T00:00:00Z"
  }
}
```

### Update Resource

Update an existing resource.

<ParamField body="name,type" path="id" query="none">

<ParamFieldObject name="id" type="string" required>
  Resource ID
</ParamFieldObject>

<ParamFieldObject name="name" type="string">
  Updated name of the resource
</ParamFieldObject>

<ParamFieldObject name="type" type="string">
  Updated type of the resource
</ParamFieldObject>

</ParamField>

<CodeGroup>
```bash cURL

  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \

    "name": "Updated Resource",
    "type": "updated"

```

```javascript JavaScript

  },

    name: 'Updated Resource',
    type: 'updated'
```
