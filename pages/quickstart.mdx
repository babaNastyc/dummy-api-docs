# Quickstart

This quickstart guide will help you get started with the Dummy API in just a few steps.

## Prerequisites

- An API key (you can get one from the dashboard)
- cURL or your favorite HTTP client
- Basic knowledge of REST APIs

## Step 1: Get Your API Key

First, you'll need to obtain an API key. For this dummy API, you can use:

```bash
API_KEY="dummy_api_key_12345"
```

## Step 2: Make Your First Request

Let's make a simple GET request to test the API:

<CodeGroup>
```bash cURL
curl -X GET https://api.example.com/v1/status \
  -H "Authorization: Bearer dummy_api_key_12345" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/status', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer dummy_api_key_12345',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer dummy_api_key_12345',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.example.com/v1/status', headers=headers)
print(response.json())
```

</CodeGroup>

**Expected Response:**

```json
{
  "status": "ok",
  "message": "API is running",
  "version": "1.0.0"
}
```

## Step 3: Create a Resource

Now let's create a resource:

<CodeGroup>
```bash cURL
curl -X POST https://api.example.com/v1/resources \
  -H "Authorization: Bearer dummy_api_key_12345" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Resource",
    "type": "example"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/resources', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer dummy_api_key_12345',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Resource',
    type: 'example'
  })
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

**Expected Response:**

```json
{
  "id": "res_123",
  "name": "My Resource",
  "type": "example",
  "created_at": "2025-01-01T00:00:00Z"
}
```

## Step 4: Retrieve Your Resource

Get the resource you just created:

<CodeGroup>
```bash cURL
curl -X GET https://api.example.com/v1/resources/res_123 \
  -H "Authorization: Bearer dummy_api_key_12345"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/resources/res_123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer dummy_api_key_12345'
  }
});

const data = await response.json();
console.log(data);
```

</CodeGroup>
