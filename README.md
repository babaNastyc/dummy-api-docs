## Dummy API Docs – Mintlify Learning Repository

This repository is a simplified Mintlify documentation setup for learning Mintlify deployment from a personal account.

### Structure

- **Config**: `mint.json` – Mintlify configuration file
- **Content**: `pages/` – Documentation pages in MDX format
  - `introduction.mdx` – Overview and getting started
  - `quickstart.mdx` – Quick start guide
  - `api/*` – API reference documentation

### Getting Started with Mintlify

1. **Create a Mintlify account** at [mintlify.com](https://www.mintlify.com)
2. **Connect your repository** – Link this repository to your Mintlify account
3. **Deploy** – Mintlify will automatically build and deploy your documentation
